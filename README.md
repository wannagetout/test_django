# test_django

## Тестовое задание для OWL Agency

1. Установка

В папке с приложением

```bash
python3 -m venv evn
```
```bash
source env/bin/activate
```
```bash
pip install -r requirements.txt
```
```bash
python3 manage.py migrate
```


2. Запуск приложения

```bash
python3 manage.py runserver
```
